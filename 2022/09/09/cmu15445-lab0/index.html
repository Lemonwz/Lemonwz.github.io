cmu15445 project#0 - C++ Primer

Project #0: C++ Primer

这部分实验要求补充Matrix、RowMatrix和RowMaxtrixOperations三个类中对应函数的代码，RowMatrix类继承自Matrix类，表示一个二维矩阵；RowMatrixOperation类提供RowMatrix对象的矩阵加法和乘法运算

实验地址：https://15445.courses.cs.cmu.edu/fall2021/project0/

Matrix：

拷贝构造函数，linear_是一个将矩阵元素平铺的一维动态数组，长度为给定的行数和列数的乘积

成员函数参数列表后的const的作用是修改隐式this指针的类型，默认情况下this的类型是指向类类型非常量版本的常量指针（即类一旦实例化一个对象后，this就指向这个对象且不能改变指向，但这个对象本身是可以改变的），添加const后，this便是一个指向类类型常量版本的常量指针，即对象本身也是不可改变的（即const成员函数不能改变调用它的对象的内容）

通过在虚函数参数列表后添加=0即可将虚函数声明为纯虚函数，从而告诉用户这个虚函数是没有意义的，它仅仅是一个接口，由派生类来具体定义它的行为

通过对具有合成版本的成员函数（默认构造函数或拷贝构造函数）使用=default来显式地要求编译器为指定的构造函数生成默认构造函数（编译器在发现类不包含任何构造函数时才会生成一个默认构造函数）

RowMatrix：

派生类不能继承基类的构造函数，但可以在派生类的构造函数中调用基类的构造函数来完成初始化，这里通过: Matrix<T>(rows, cols)调用对应的基类拷贝构造函数（语法规定创建派生类对象时必须调用基类的默认构造函数，也可以在派生类构造函数中显式指定要调用的基类构造函数），随后再初始化派生类自己的成员变量，这里data_是一个二维数组，存放二维矩阵的元素值

获取矩阵行数和列数，派生类通过override关键字显式注明该成员函数覆盖了对应的虚函数（函数声明必须一致）

获取和设置矩阵对应位置的元素值，如果小于0或超出了矩阵大小则抛出异常

通过一维vector填充矩阵（行优先），如果vector长度和矩阵元素个数不一致则抛出异常

最后通过析构函数释放分配的内存空间
